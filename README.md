# Играть Пасьянс

Играть Пасьянс — это браузерная версия классической карточной игры Solitaire, доступная прямо в браузере без установки.

🌐 Website  
https://играть-пасьянс.рф/

## About

Проект создан для игроков, которые хотят быстро начать играть в пасьянс прямо в браузере.

Сайт предоставляет простой интерфейс и классическую механику пасьянса, позволяя начать игру сразу без регистрации или загрузки приложений.

## Gameplay

Игрок раскладывает карты по правилам классического пасьянса:

- карты раскладываются по мастям
- последовательности собираются по убыванию
- цель — разложить всю колоду

Игра сочетает стратегию, внимание и расслабляющий игровой процесс.

## Features

- Классический пасьянс в браузере
- Быстрый старт игры
- Работает на компьютере и мобильных устройствах
- Простая и понятная игровая механика
- Интерфейс, ориентированный на быстрый доступ к игре

## Philosophy

Проект следует идее простых веб-игр, которые можно открыть и начать играть за несколько секунд.

Без установки, без регистрации — просто игра.
